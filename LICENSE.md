## Tecnologias Utilizadas

Todas as tecnologias utilizadas neste projeto são frameworks nativos da Apple:
- SwiftUI
- Combine
- MapKit
- CoreLocation
- Foundation
- URLSession

**Nenhuma biblioteca de terceiros foi utilizada.**

## Sobre o Projeto

CidadeViva é um aplicativo de demonstração que mostra as melhores práticas de desenvolvimento iOS moderno:
- Clean Architecture
- MVVM Pattern
- Protocol-Oriented Programming
- Async/Await
- SwiftUI
- Dados mockados (não conecta a APIs reais)

## Limitações

⚠️ **IMPORTANTE:**
- Os dados são mockados e fictícios
- Não há conexão com servidores reais
- As câmeras e alertas são apenas exemplos
- As coordenadas são genéricas de São Paulo/SP
